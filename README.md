# 抚幼校园圈（PWA 版）

抚州幼儿师范高等专科学校校园生活服务平台的 PWA 版本，基于 Vue 3 + Vite 构建。支持离线访问，可添加到手机桌面像原生 APP 一样使用。

## 功能模块

- **首页** — 校园公告、功能导航入口
- **论坛** — 校园话题讨论、帖子发布与评论
- **吐槽墙** — 匿名倾诉、情感交流
- **校园服务** — 兼职、快递代取、外卖代取、二手交易、广告发布
- **用户中心** — 学生认证、个人信息、我的发布
- **管理后台** — 内容审核、用户管理、举报处理、敏感词管理

## 技术栈

- Vue 3 + TypeScript + Vue Router + Pinia
- Vite + vite-plugin-pwa
- SCSS 预处理器
- Mock 数据（纯前端演示）

## 本地开发

```bash
npm install
npm run dev
```

## 构建生产版本

```bash
npm run build
npm run preview
```

## PWA 安装方式

### 安卓（Chrome/Edge）
1. 用浏览器打开部署后的网址
2. 底部会自动弹出"安装"提示，点击即可
3. 或在菜单中选择"添加到主屏幕"

### iOS（Safari）
1. 用 Safari 打开网址
2. 点击底部分享按钮
3. 选择"添加到主屏幕"

### 电脑（Chrome/Edge）
1. 地址栏右侧会出现安装图标，点击安装
2. 或在菜单中选择"安装抚幼校园圈"

## 离线使用

首次访问后，页面和资源会被缓存到本地。后续即使断网，已访问过的页面仍可正常浏览。
